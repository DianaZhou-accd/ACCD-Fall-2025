ESP32 + Processing, Sound + Visual System by Diana Zhou

Collaborative Button Sampler

An interactive system where four physical buttons trigger sound samples and generate simple butterfly-silhouette visuals. Designed for multiple people to play together.

Summary

I created an interactive system using an ESP32 and Processing. The ESP32 sends data from four buttons to a Processing sketch through serial communication. Each button triggers a sound and also creates a visual response on the screen.

I chose buttons as sensors because they are very intuitive: press once, one action happens. This clear interaction works well for rhythm and group interaction, and it does not require any learning or explanation.

In Processing, I used a sound library to play audio files. Each button plays a different sound. Multiple sounds can overlap, slowly building rhythm and structure. The system works like a simple drum pad or sampler.

For the visuals, I use butterfly silhouettes as symbols. Pressing a button creates a butterfly, and other buttons change the butterfly's size and movement speed. These visual changes represent sound intensity, density, and rhythm.

Designed for Multiple People

This system is designed for multiple people. Different users can control different buttons at the same time and collaboratively “play” the system together. The interface is simple and easy to join, even for first-time users.

Controls

  • BTN1 — Spawn a butterfly (sound trigger)
  • BTN2 — Increase butterfly size (intensity)
  • BTN3 — Increase butterfly speed (density)
  • BTN4 — Reset the system
