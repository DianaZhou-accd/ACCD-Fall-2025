Rotation

Time-based grid oscillation study (p5.js). Below is the step-by-step process documentation with iterations.

Process Documentation (6 Steps)

Step 1 — Basic Grid Structure

Built a simple grid using for-loops to establish a stable structure before introducing motion or time.

Step 2 — Parameterized Grid (Function & Variable)

Refactored the grid into a function and controlled its density with a variable, making the structure reusable and adjustable.

Step 3 — Time & Oscillation (sin + time)

Introduced a time variable and used a sine function to oscillate the number of grid lines, creating a rhythmic “breathing” effect.

Step 4 — Rotation (push/pop transformation)

Applied rotation to the oscillating system using push/pop to isolate transformations and explore spatial motion.

Step 5 — Layering (second grid with different transform)

Duplicated the grid as a second layer with translate + counter-rotation, creating interference between two similar systems.

Step 6 — Anchor Shape (readability)

Added a centered rectangle as a visual anchor to make the oscillation and layered rotations easier to read.
